SafiServer FreePBX Module Install Guide


You can download a copy of the SafiServer FreePBX Module here.


Installing Safi Systems FreePBX Module

1) Go to Module Admin in the FreePBX web admin.
2) Click Upload Module
3) Browse to the saflets.tar.gz file and click upload
4) Go back to the Module Admin and select Saflets under Inbound Call Control
5) Select the Install radio button
6) Press the Process button (there's one at the top right of the page) then confirm that you want to install



Upgrading the Saflets FreePBX Module

1) Go to Module Admin in the FreePBX web admin.
2) Click Upload Module
3) Browse to the saflets.tar.gz file and click upload
4) Go back to the Module Admin and select Saflets under Inbound Call Control
5) Select the Upgrade radio button
6) Press the Process button then confirm that you want to upgrade
