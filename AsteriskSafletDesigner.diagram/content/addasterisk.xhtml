Setting up SafiServer: Install Linux Version of SafiServer
  


1) Right-click the Asterisk Server in the Navigator and select or ...

Click here to open the Asterisk Configuration

You should see the following popup:



2) Fill out the Asterisk configuration form

     Unique Server Name: A unique name used to identify your Asterisk PBX

     Host(IP): IP address or hostname of your Asterisk PBX.

     Manager Name/Password: The login and password set up earlier in this walkthrough


3) Press OK to save the changes.
