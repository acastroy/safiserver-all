Now that the Asterisk server is set up to accept connections from the Debug SafiServer, you will need to set that login/password in SafiWorkshop. You can open the Debug SafiServer Manager by double clicking on the Local(Debug) Asterisk server node (under the Debug: node) in the the navigator or ...

Click here to open the Asterisk Configuration

You should see the following popup:


Fill in the Host(IP) field with the IP address of your Asterisk PBX.

Fill in the Manager Name and Manager Password with the values you set in step 2.

Press OK to save the changes.
