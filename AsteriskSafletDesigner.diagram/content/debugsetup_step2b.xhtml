2) Configure Asterisk PBX (FreePBX w/ Saflets module):
If you are running Asterisk with FreePBX you can quickly configure the settings needed using the SafiServer FreePBX Module.

After you've installed the module (click here for the install guide) go to the SafiServer menu option in the FreePBX web administrator:



Select Add SafiServer Entry on the right and you should see the following form:



Fill in the values as follows:

SafiServer Name : This can be any name you want (example: Workshop Dev)

SafiServer IP Address : Set this to the IP address that your SafiWorkshop is running on (the value set in the first step)

SafiServer FastAGI Port : Change this to 3573 (4573 is for use with a production SafiServer)

SafiServer IP Netmask : Set this the to subnet associated with your the IP (SafiServer IP Address)

Manager Login : Set this to whatever logon you wish to use (example: safi)

Manager Password : Set this to whatever password you wish to use (example: 1234)

Is debug server? : Leave this checked


Here's an example of what the form should look like:


Press Submit Changes then press the Apply Configuration Changes link on the top of the page
