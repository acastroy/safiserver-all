2) Configure Asterisk PBX (configuration files):
Next, you need to set up the Asterisk PBX that you want to use with the debug server. On the Asterisk server you will need to modify the manager.conf file to allow the Debug SafiServer to connect. The manager.conf is located by default at /etc/asterisk/manager.conf

Here's an example addition to the manager.conf configuration file that all allows a SafiServer with the IP address of 192.168.0.10, user of safi, and with the password of 1234 access:
;
; SafiServer connection
;
[safi]
secret=1234
deny=0.0.0.0/0.0.0.0
permit=192.168.0.10/255.255.255.0
read = system,call,log,verbose,command,agent,user
write = system,call,log,verbose,command,agent,user

NOTE: Make sure to use the IP address/subnet (the permit line) of the machine your SafiWorkshop is installed on; otherwise, the debug server will not be able to make the connection.

After the manager.conf has been modified (and saved), you will need to run the reload command in Asterisk CLI.
