Setting up SafiServer: Install Linux Version of SafiServer
  
1) Untar (or unzip) the file into a directory of your choice.

Example:
# tar zxfv SafiServer32-1.x.x.tar.gz
2) Navigate to the SafiServer directory.

Example:
# cd SafiServer
3) Run the install script file.

Example:
# sudo ./installserver.sh
4) Configure server settings

On the first run of the installserver.sh shell script; the following configuration page will be displayed:



You will be prompted to enter the following values (press enter to use the values in parenthesis):
  • SafiServer bind IP: This is the local IP address that the SafiServer will use to bind to
  • FastAGI port: The TCP port on which to run the SafiServer FastAGI server on
  • sa password: This is the system administrator password for the SafiServer database server
