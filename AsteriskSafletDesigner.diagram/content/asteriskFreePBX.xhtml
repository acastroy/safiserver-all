Setting up SafiServer: Configuring SafiServer module in FreePBX
  


1) Navigate to the SafiServer menu option in the FreePBX web administrator:



2) Select Add SafiServer Entry on the right and you should see the following form:



3) Fill in the values as follows:

     SafiServer Name : This can be any name you want (example: MySafiServer)

     SafiServer IP Address : Set this to the IP address (or host name) of your SafiServer

     SafiServer FastAGI Port : Port that Asterisk will use when making an AGI call (4573 is default)

     SafiServer IP Netmask : Set this the to subnet associated with the SafiServer IP Address

     Manager Login : Set this to whatever logon you wish to use (example: safi)

     Manager Password : Set this to whatever password you wish to use (example: 1234)

4) Press Submit Changes

5) Press the Apply Configuration Changes link on the top of the page
