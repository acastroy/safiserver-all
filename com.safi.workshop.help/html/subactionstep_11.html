ActionStep

WaitForDigit

Description

Waits for one of the defined DTMF keys to be pressed before proceeding to the next ActionStep. This is a single key press and does not store the pressed key in a variable.

Attributes

Name Default Description
Accepted Digits 012345678# DTMF keys that will cause the step to proceed
Call1* Call1 The object representing the call to wait for keypress
Name* WaitForDigit Name used in the SafiWorkshop for the ActionStep
Timeout -1 Number of milliseconds to wait before proceeding to the next step (-1 to wait indefinitely)
