ActionStep

ExtensionSpy

Description

Listens in on a given channel/extension. This listening feature remains active until through the duration of the listeners call and will wait for a connection to be made to the given extension.

While monitoring the following key presses may be used:

# : Stepwise volume adjustment (-4 to 4)
* : Switch to another channel

Attributes

Name Default Description
Beep true ( true | false ) If set to true the system will play a tone and say the channel name when listening starts on a channel.
Call1* Call1 Call object
Channel Name
Context Used to further narrow down the extension by context
Extension Extension to listen in on
Group If set, only listens to channels where the channel variable SPYGROUP is set to the value entered for this property.
NOTE: SPYGROUP} can contain a list of values, separated by a :.
Name* ExtensionSpy Name used in the SafiWorkshop for the ActionStep
Private Whisper Enabled false ( true | false ) Enables "private whisper mode". The "spying" channel can whisper to the spied-on channel, but cannot listen.
Record Filename Prefix If present; records the listening session to the spool directory.
Spy Bridged Only ( true | false ) Only listens to channels which belong to a bridged call.
Volume Sets the initial volume. The value may be between -4 and 4.
Whisper Enabled false ( true | false ) Enables "whisper" mode. Lets the spying channel talk to the spied-on channel.
