ActionStep

GetColValue

Description

Gets the value of a specified column in a result set and assigns it to a variable.

Attributes

Name Default Description
Column* The name of the column in the result set to get the value of
Get As Datatype Test The datatype to cast the column value to
Name GetColValue The Name of the ActionStep (used for reference in SafiWorkshop)
Result Set* null The result set to use (selected from drop-down list)
Variable Name* The name of the variable to store the column value in. The variable selection tool can be invoked by pressing the "..." button.
