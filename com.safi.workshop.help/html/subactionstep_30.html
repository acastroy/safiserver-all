ActionStep

VoicemailMain

Description

Enters the Asterisk voicemail system for the checking of voicemail. The mailbox can be passed as a property: Mailbox, which will stop the voicemail system from prompting the user for the mailbox.

If the mailbox property Skip Password Check is set to false; the password check will be skipped.

VoicemailMain Menu
  • 1 Read voicemail messages
    • 3 Advanced options
      • 1 Reply
      • 2 Call back(1)
      • 3 Envelope
      • 4 Outgoing call(1)
      • 5 Send Message (only available if sendvoicemail=yes in voicemail.conf)
    • 4 Play previous message
    • 5 Repeat current message
    • 6 Play next message
    • 7 Delete current message
    • 8 Forward message to another mailbox
      • 1 Use Voicemailnumber (only available if usedirectory=yes in voicemail.conf)
      • 2 Use Voicemail Directory (only available if usedirectory=yes in voicemail.conf)
    • 9 Save message in a folder
      • 0 Save in new Messages
      • 1 Save in old Messages
      • 2 Save in Work Messages
      • 3 Save in Family Messages
      • 4 Save in Friends Messages
    • * Help; during msg playback: Rewind
    • # Exit; during msg playback: Skip forward
  • 2 Change folders
    • 0 Switch to new Messages
    • 1 Switch to old Messages
    • 2 Switch to Work Messages
    • 3 Switch to Family Messages
    • 4 Switch to Friends Messages
  • 3 Advanced Options
    • 5 Send Message (only available if sendvoicemail=yes in voicemail.conf)
      • 1 Use Voicemailnumber (only available if usedirectory=yes in voicemail.conf)
      • 2 Use Voicemail Directory (only available if usedirectory=yes in voicemail.conf)
  • 0 Mailbox options
    • 1 Record your unavailable message
    • 2 Record your busy message
    • 3 Record your name
    • 4 Record your temporary message (new in Asterisk v1.2)
    • 5 Change your password
    • * Return to the main menu
  • * Help
  • # Exit


After recording a message (incoming message, busy/unavailable greeting, or name)
  • 1 Accept
  • 2 Review
  • 3 Re-record
  • 0 Reach operator(1) (not available when recording greetings/name)



See the following for further information reguarding the Asterisk voicemail system configuration file (voicemail.conf): http://www.voip-info.org/wiki/index.php?page=Asterisk+config+voicemail.conf

Attributes

Name Default Description
Call1 Call1 The object representing the call to transfer to voicemailmain
Mailbox Mailbox formatted like: mailbox@context
Name VoicemailMain Name used in the SafiWorkshop for the ActionStep
Skip Password Check false ( true | false ) Skip checking the passcode for the mailbox.
