ActionStep

Dictate

Description

Asterisk Application: Virtual Dictation Machine

Options:
  • 1 to toggle record and playback modes
  • 0 for help
  • * pause/unpause
  • # to enter a new filename

Playback Mode Options
  • 2 to toggle fast playback (speed 1x, 2x, 3x, 4x)
  • 7 to seek backwards a few frames
  • 8 to seek forwards a few frames

Record Mode Options
  • 8 to erase the whole file and start again

Attributes

Name Default Description
Call1 Call1 The object representing the call to run Dictate application on
Directory Directory on Asterisk server to store dictation files (if blank /tmp/dictations is used)
Filename ( Optional ) Name of file to use for dictation
Name Dictate Name used in the SafiWorkshop for the ActionStep
