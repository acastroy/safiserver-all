ActionStep

RecordFile

Description

Records from the current channel to a sound file saved with the given filename. The format parameter specifies the sound format and the extension of the file. If you don't specify a full path, the file will be stored in the /var/lib/asterisk/sounds directory. If a file with the same name and extension already exists, it will be overwritten.

Attributes

Name Default Description
Beep true ( true | false ) Play a beep before recording
Call1* Call1 The object representing the call to record
Escape Digits # Key user can press to end this ActionStep
File* File to record call to. If no path is defined the file will be stored in the /var/lib/asterisk/sounds directory.
Format wav One of the following supported formats: sln, g723, g729, gsm, h263, ulaw, alaw, vox, wav, WAV (WAV is the GSM format of wav files)
Max Silence 0 Seconds of silence allowed before the recording is stopped. If missing or 0, silence detection is disabled.
Name* RecordFile Name used in the SafiWorkshop for the ActionStep
Offset 0 (samples) If provided this step will try to the offset without exceeding the end of the file.
Timeout -1 Maximum recording duration in seconds. If missing or 0, there is no maximum
