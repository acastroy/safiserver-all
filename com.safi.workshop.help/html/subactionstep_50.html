ActionStep

ChanSpy

Description

Listen in on a call by specifying a Channelname Prefix. This adds the ability to "spy" (or monitor) any bridged call, this includes VoIP only calls.

While monitoring the following key presses may be made:
# : Cycles the volume level.
* : Stop the current monitoring session and looks for another channel to listen in on.

Dialing a series of digits followed by # appends to the end of the set channel name prefix
Example:
With the property setting: Channel Name Prefix set to "Agent"
Dial 1234# while monitoring to jump to channel Agent/1234

Attributes

Name Default Description
Beep true ( true | false ) If false; Don't play a beep when beginning to spy on a channel, or speak the selected channel name.
Call1 Call1 The object representing the call to use for playing the monitor sound to
Channel Name Prefix If specified, only channel names *beginning* with that string will be scanned.
Group Match only channels where their ${SPYGROUP} variable is set to contain 'grp' in an optional : delimited list.
Name ChanSpy Name used in the SafiWorkshop for the ActionStep
Private Whisper Enabled false ( true | false ) Enable 'private whisper' mode, so the spying channel can talk to the spied-on channel but cannot listen to that channel.
Record Filename Prefix Record the session to the monitor spool directory. An optional base for the filename may be specified. The default is 'chanspy'.
Spy Bridged Only false ( true | false ) Only listen to channels involved in a bridged call.
Volume 0 Adjust the initial volume in the range from -4 to 4. A negative value refers to a quieter setting.
