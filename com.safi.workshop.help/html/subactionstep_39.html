ActionStep

SayNumber

Description

Plays back the number specified

Example: The input: 312 will playback: "three-hundred twelve".

Attributes

Name Default Description
Call1* Call1 The object representing the call to play to
Escape Digits # Key (or keys) user can press to skip this ActionStep
Name* SayNumber Name used in the SafiWorkshop for the ActionStep
Number* Number to say
