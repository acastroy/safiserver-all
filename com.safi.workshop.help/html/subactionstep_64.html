ActionStep

GetAstVar

Description

Returns the value held in a give Asterisk channel or global variable.

Attributes

Name Default Description
Call1* Call1 Call object
Name* GetAstVar Name used in the SafiWorkshop for the ActionStep
Variable* The name of the variable to retrieve the value of
[o] Assign To Var* variableValue The SafiServer variable to save the value of the Asterisk variable in
