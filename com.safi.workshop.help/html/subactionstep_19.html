ActionStep

StreamAudioExtended

Description

Plays a sound file (do not include the filename extension) with fast forward, rewind and stop controls.

By Default:
[1] = Rewind
[2] = Pause
[3] = Fast Forward

Attributes

Name Default Description
Call1* Call1 The object representing the call to play to
Escape Digits # Key user can press to skip this ActionStep
Filename* Name of file to play (with out the extension)
Forward Digit 3 Fast-forward when this DTMF digit is received
Name* StreamAudioExtended Name used in the SafiWorkshop for the ActionStep
Offset 0 This is number of milliseconds to skip when rewinding or fast-forwarding
Pause Digit 2 Pause playback when this DTMF digit is received
Rewind Digit 1 Rewind when this DTMF digit is received
