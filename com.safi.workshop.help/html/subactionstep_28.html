ActionStep

SetCallerId

Description

Sets the calls CallerID (or ANI) information.

Attributes

Name Default Description
Call1* Call1 The object representing the call to set caller id
Caller Id* Caller Id to set on the given call. This can be a numeric or text value. To set both the caller name and number; use the following format: Caller Name <callernumber> (i.e. Joe Bob<5035551212>)
Name* SetCallerId Name used in the SafiWorkshop for the ActionStep
