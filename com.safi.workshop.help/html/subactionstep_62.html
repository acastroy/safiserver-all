ActionStep

Extension

Description

ActionStep for transfering a call to a given local extension. Has the following exit paths for further call handling:
  • bad ext: Extension was not found in Asterisk
  • dnd: Extension set to do not disturb
  • cancelled:
  • no answer: Call was not answered in the given timeout
  • busy: Channel is currently in use


The default path is taken after the extension called is hung up. This can be used for post call processing.

Attributes

Name Default Description
Call1 Call1 The object representing the call to be transfered
Call2 Call2
Context Asterisk context
Extension The local extension to transfer the call to
Name* Extension Name used in the SafiWorkshop for the ActionStep
Priority 1 Asterisk priority
Timeout(sec) 0 The alert timeout
