ActionStep

MixMonitor

Description

Turns on the recording(native) of the call. The output file will, by default, be located in the /var/spool/asterisk/monitor directory if a full path is not specified. The default path can be changed by updating the astspooldir setting in asterisk.conf.

Attributes

Name Default Description
Append Mode false ( true | false ) If true, MixMonitor will append to the recording file instead of overwriting it.
Call1 Call1 Object representing the call to record
Filename Name of file to record (both filename and full path are accepted) with extension
Name MixMonitor null
Record Only When Bridged false ( true | false ) Only save audio to the file while the channel is bridged. (does not include conferences)
