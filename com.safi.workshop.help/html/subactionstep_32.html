ActionStep

ExecuteApplication

Description

Executes an Asterisk application (or command).

For a list of Asterisk applications, click here: http://www.voip-info.org/wiki/index.php?page=Asterisk+-+documentation+of+application+commands

Attributes

Name Default Description
Application Asterisk application (also refered to as commands)
Arguments Arguments to pass
Call1 Call1 The object representing the call to transfer apply run application on
Name ExecuteApplication Name used in the SafiWorkshop for the ActionStep
