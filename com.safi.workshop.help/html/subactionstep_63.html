ActionStep

GetCallInfo

Description

Returns information about the call object pass in (Call1).

Attributes

Name Default Description
Call1* Call1 Call Object
Name* GetCallInfo Name used in the SafiWorkshop for the ActionStep
[o] Account Code Var accountCode Variable used to store any account code associated with the call
[o] Caller Id Name Var callerIdName Variable used to store the caller ID name
[o] Caller Id Number Var callerIdNumber Variable used to store caller ID number (ANI)
[o] Channel Name Var channelName Variable used to store the name of the channel associated with the call
[o] Context Var context Variable used to store Asterisk context
[o] Extension Var extension Variable used to store the extension associated with the call
[o] Priority Var priority Variable used to store the Asterisk priority
[o] State Var state Variable used to store the state of the call
[o] Unique Id Var uniqueId Variable used to store the unique ID created by Asterisk for each call instance
