ActionStep

MoveToInsertRow

Description

Moves the cursor position within the selected result set to the insert row. The insert row is a special row associated with an updatable result set. It is essentially a buffer where a new row may be constructed by using the SetColValue or SetColValues ActionSteps prior to inserting the row into the result set. All of the columns in a result set must be given a value each time this method is called before executing the InsertRow ActionStep. Values cannot be read from the result set while positioned on the insert row.

Attributes

Name Default Description
Name* null The unique name of the ActionStep (used for reference in SafiWorkshop)
Result Set* null The result set to use (selected from drop-down list)
