ActionStep

ExecuteUpdate

Description

Executes the given Query object, which may be an INSERT, UPDATE, or DELETE statement or an SQL statement that returns nothing, such as an SQL DDL statement. Optionally, an integer variable (parameter "Variable Name") may be provided to store the number of rows updated, if any.

Attributes

Name Default Description
Name ExecuteUpdate The name of this Actionstep instance
Query null The Query object to be executed.
