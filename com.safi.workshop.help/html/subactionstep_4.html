ActionStep

Sleep

Description

Waits for a given amount of time (in milliseconds) before continuing on to the next ActionStep

Attributes

Name Default Description
Call1 Call1 The object representing the call to apply sleep to
Duration 0 The number of seconds to wait before contenuing to the next ActionStep
Name Sleep Name used in the SafiWorkshop for the ActionStep
Native false ( true | false ) If this is set to true; Asterisk Wait command will be used
