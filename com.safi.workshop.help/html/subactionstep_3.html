ActionStep

Transfer

Description

Transfers call to the specified extension, phone number, or Asterisk context. If advanced post call processing is needed; see the ActionStep Extension

Attributes

Name Default Description
Call1* Call1 The object representing the call to be answered
Call2 New call object
Context Asterisk context to transfer the call into
Extension* Extension or phone number to transfer the call to
Name* Transfer Name used in the SafiWorkshop for the ActionStep
Priority 1 Asterisk context priority for the transfer
