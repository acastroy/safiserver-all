ActionStep

MoveToRow

Description

Moves the cursor to the row number specified by the 'Row Num' parameter in the selected result set object. If the row number is positive, the cursor moves to the given row number with respect to the beginning of the result set. The first row is row 1, the second is row 2, and so on.

If the given row number is negative, the cursor moves to an absolute row position with respect to the end of the result set. For example, using a 'Row Num' value of -1 positions the cursor on the last row; using a 'Row Num' value of -2 moves the cursor to the next-to-last row, and so on.

An attempt to position the cursor beyond the first/last row in the result set causes the ActionStep to take the 'out of bounds' output path.

Attributes

Name Default Description
Name* MoveToRow The unique name of the ActionStep
Result Set* null The result set object (selected from drop down list)
Row Num* null DynamicValue containing the number of the row to which the cursor should move. A positive number indicates the row number counting from the beginning of the result set; a negative number indicates the row number counting from the end of the result set
