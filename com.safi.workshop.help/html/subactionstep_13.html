ActionStep

SayAlpha

Description

Says a string of letters, numbers, and/or selected other characters.

Attributes

Name Default Description
Call1* Call1 The object representing the call to play to
Escape Digits # Key user can press to skip this ActionStep
Name* SayAlpha Name used in the SafiWorkshop for the ActionStep
Text* String of letters, numbers, and/or other characters
Other Caracters:
  • ! exclamation point
  • @ at
  • # pound
  • $ dollar
  • * star
  • - dash
  • + plus
  • = equals
  • / slash
  • ' ' (an empty space) space
  • . dot
