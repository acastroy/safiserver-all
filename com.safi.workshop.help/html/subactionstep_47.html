ActionStep

OriginateCall

Description

Places a call and creates a call object

Attributes

Name Default Description
false ( true | false )
Account Account code for the call
Application Application to use on connect (use Data for parameters)
Async false ( true | false ) For the origination to be asynchronous (allows multiple calls to be generated without waiting for a response)
Caller Id CallerID to use for the call
Calling Presentation 1
Channel Channel on which to originate the call (The same as you specify in the Dial application command)
Context Context to use on connect (must also fill in Extension and Priority Properties in order to use)
Data Data if Application parameter is used
Extension Extension to use on connect (must also fill in Context and Priority Properties in order to use)
Name OriginateCall Name used in the SafiWorkshop for the ActionStep
New Call1 New Call Object
Priority 1 Priority to use on connect (must also fill in Context and Extension Properties in order to use)
Timeout 0 Timeout (in milliseconds) for the connection to happen
Variables Channels variables to set (max 32). Variables will be set for both channels (local and connected).
