ActionStep

Monitor

Description

Starts the recording of a given call. If you don't specify a full path, the file will be stored in the "monitor" subdirectory of the path specified with astspooldir in asterisk.conf (by default this will be /var/spool/asterisk/monitor).

Attributes

Name Default Description
Call1 Call1 The object representing the call to record
Filename Prefix The base filename to use when saving the sound files. If not supplied, the default basename is constructed on the channel name plus a number. The default location for saved monitor files is the /var/spool/asterisk/monitor directory.
Format wav The sound file format to save in, which will be also used as the filename extension.
Mix false ( true | false ) If true, then when recording finishes, Asterisk will execute a unix program to combine the two sound files into a single sound file.
Name Monitor Name used in the SafiWorkshop for the ActionStep
