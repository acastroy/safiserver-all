ActionStep

PromptGetDigits

Description

Plays an audio file while waiting for a defined number of DTMF key presses

Attributes

Name Default Description
Call1* Call1 The object representing the call to wait for keypress
Filename Name of file to play to call
Max Digits 1 Number of digits to wait for before continuing
Name* PromptGetDigits Name used in the SafiWorkshop for the ActionStep
Timeout 0 Timeout in seconds before preceeding to the next step
Used Buffered Digits false ( true | false ) If set to true; use key presses from previous StreamAudio/MultiStreamAudio ActionSteps. NOTE: only valid Escape Digits will result in buffered digits being passed.
[o] Variable Name capturedDigits Name of output variable to store the captured keypresses in
