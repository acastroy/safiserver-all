ActionStep

IncomingCall

Description

Initiator that is launched by an FastAgi statement in extensions.conf (or an included file) on an Asterisk server.


extensions.conf syntax:
exten = extension,1,Agi(agi://SafiServer Address/handlerEngine.agi?saflet=Project Name/Saflet Name)


Example:
exten = 1111,1,Agi(agi://192.168.0.10/handlerEngine.agi?saflet=project1/callflow1)
