ActionStep

SetMusicOnHold

Description

Sets the default class for music on hold for a call. When music on hold is activated, this class will be used to select which music is played.

Click here for more information on configuring music on hold

Attributes

Name Default Description
Call1 Call1 Call to set the hold class of
Hold Class The name of the hold class to use for the given call
Name SetMusicOnHold Name used in the SafiWorkshop for the ActionStep
