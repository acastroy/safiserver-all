ActionStep

PlayDTMF

Description

Play DTMF tones to the specified call. This can be useful for auto-navigating external IVR systems.

Attributes

Name Default Description
Call1* Call1 The object representing the call to play to
Digits* ( 0123456789*# ) DTMF digit (or digits) to play. Commas may be used to pause in between digit (i.e. 200,1,,3)
Name* PlayDTMF Name used in the SafiWorkshop for the ActionStep
