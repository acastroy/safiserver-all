ActionStep

InsertRow

Description

Inserts the contents of the insert row into the selected result set object and into the database. A typical sequence of ActionSteps might be, RunQuery -> MoveToInsertRow -> SetColValues -> InsertRow. Note that no changes to the underlying database are committed until the InsertRow ActionStep completes execution.

Attributes

Name Default Description
Name* InsertRow The Name of the ActionStep (used for reference in SafiWorkshop)
Result Set* null The result set to use (selected from drop-down list)
