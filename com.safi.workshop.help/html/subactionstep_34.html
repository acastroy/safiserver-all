ActionStep

PlaceCall

Description

Places a call from the system and creates a new call object. This allows the handling of each call (inbound and outbound) separately.

Attributes

Name Default Description
Caller ID Caller ID to use on the placed call
Context Context to place call from. In FreePBX installs the context "from-pstn" works for most situations
Extension* Extension to place call to
Name PlaceCall Name used in the SafiWorkshop for the ActionStep
New Call1 Call2 New call object
Timeout(ms) 0 Time to wait for call to be answered in milliseconds
