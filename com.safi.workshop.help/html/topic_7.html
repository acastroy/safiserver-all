SafiServer


The SafiServer is the core component of the Safi Systems architecture.   Based on Java technology, the SafiServer can handle many threads of operation at once.  SafiServer runs applications created using SafiWorkshop called Saflets.  Currently these Saflets are focused around call flow type functionallity, but these applications can be used for any flow based situation.

There is a limited version of the SafiServer embedded inside the Saflet designer: SafiWorkshop.  This allows SafiWorkshop the ability to test and debug Saflets locally (on the machine SafiWorkshop is installed on) before deploying to a production SafiServer.

The SafiServer has a telephony component to it that allows it to connect and control PBX devices (currently Asterisk Opensource PBX).  This allows for the centralized control of callflow / IVR functionallity.  SafiServer can connect to many PBX servers at once allowing for increased scalabilty.
