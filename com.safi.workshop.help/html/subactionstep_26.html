ActionStep

Choice

Description

SafiServer ActionStep used to validate a given value against a number of different cases (or choice items). If none of the case items match the value; the default path (green) is taken.

NOTE: When using scripting to derive a value for a Choice Option and the desired result needs to be an integer (i.e. a digit pressed on a key pad); you must take that numeric value and strip the floating point using the .toFixed(0) Javascript operation. Example (will output 3):
myVar = 1;
myVar2 = 2;
myVar3 = myVar + myVar2;
myVar3.toFixed(0);



Attributes

Name Default Description
Name* Choice Name used in the SafiWorkshop for the ActionStep
Value* Value to use in selecting a proper exit path. If this value matches the value in one of the Choice items,that exit path is taken.

NOTE: priority is highest at the top; the Choice ActionStep will take the exit path of the first match starting from the top down
