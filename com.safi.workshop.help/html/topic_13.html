Setting up extensions.conf


A Saflet can be invoked on a SafiServer (or SafiWorkshop) by making a FastAGI call to the server with the proper information (project name and Saflet name).


extensions.conf syntax:
exten = [extension],1,Agi(agi://[IP or FQDN of SafiServer]/safletEngine.agi?saflet=[Project]/[Saflet])

example:
exten = 1111,1,Agi(agi://192.168.0.10/safletEngine.agi?saflet=project/saflet)


If needed, you might need to specify a different port for the FastAGI connection (i.e. the FastAGI server in SafiWorkshop by default runs on port 3573 instead of the default 4573)

example:
exten = 1111,1,Agi(agi://192.168.0.10:3573/safletEngine.agi?saflet=project/saflet)
