ActionStep

SayDigits

Description

Says the digits specified one by one.

Example: the following input: 12345 will result in the system playing back: "one two three four five"

Attributes

Name Default Description
Call1* Call1 The object representing the call to play to
Digits* Digits to be read (i.e. 1234567890)
Escape Digits # Key (or keys) user can press to skip this ActionStep
Name* SayDigits Name used in the SafiWorkshop for the ActionStep
