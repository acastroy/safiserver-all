ActionStep

MeetMeAdmin

Description

Executes administrative MeetMe conference functions.

Available Commands:
eject last user (that joined the conference)
kick one user (out of conference)
kick all users (out of conference)
unlock (conference)
lock (conference)
unmute user
mute user
unmute all users (in the conference)
mute all non-admin (users in the conference)
reset one user volume
reset volume all users
lower conf speak volume (lowers the entire conference speaking volume)
raise conf speak volume (raises the entire conference speaking volume)
lower conf listen volume (lowers the entire conference listening volume)
raise conf listen volume (raises the entire conference listening volume)
lower user talk volume
raise user talk volume
lower user listen volume
raise user listen volume

Attributes

Name Default Description
Call1 Call1 The object representing the call to run MeetMeAdmin application on
Command eject last user Command to execute
Conference Number Conference number to apply command to
Name MeetMeAdmin Name used in the SafiWorkshop for the ActionStep
User User to apply command to
