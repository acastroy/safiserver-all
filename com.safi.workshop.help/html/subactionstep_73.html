ActionStep

SetColValues

Description

This ActionStep is similar to SetColValue except that it is used to update multiple columns of a given result set, rather than just one. A SetColValues ActionStep can have one or more "ColumnMapping" objects, which can be added by hovering over the ActionStep and clicking the balloon popup that appears. Each ColumnMapping object specifies a result set column with the 'Column' parameter and a value to assign to that column in the 'Value' parameter.

This ActionStep can be executed only when the cursor is on the current row or the insert row. SetColValues does not update the underlying database; instead the UpdateRow or InsertRow ActionSteps are employed to update the database.

Attributes

Name Default Description
ColumnMapping(s) null List of Column name/value pairs specifying the column to update and the value to update it with
Name SetColValues The unique name of the ActionStep (used for reference in SafiWorkshop)
Result Set* null The result set object
