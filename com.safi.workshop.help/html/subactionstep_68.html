ActionStep

RecordCall

Description

Turns on the recording of the call. The output file will, by default, be located in the /var/spool/asterisk/monitor directory if a full path is not specified. The default path can be changed by updating the astspooldir setting in asterisk.conf.

NOTE: if the same filename is used for subsequent calls, that file will be replaced

Attributes

Name Default Description
Call1* Call1 Call object to record
Filename Name of file to record (both filename and full path are accepted)
Format wav The sound file format to save in, which will be also used as the filename extension.
Mix true ( true | false ) If true, then when recording finishes, Asterisk will execute a unix program to combine the two sound files into a single sound file.
Name* RecordCall Name used in the SafiWorkshop for the ActionStep
