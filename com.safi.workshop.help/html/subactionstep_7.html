ActionStep

DebugLog

Description

Writes a message to the server console screen in the SafiWorkshop and a log file (if path is specified).

Attributes

Name Default Description
Debug Level Debug ( Debug | Warn | Error ) Added as a prefix to the log message
Log Filename Path to local file to save logs. If file does not exist; server will create one. This path is relative to the SafiServer (or if running from the debug server; the SafiWorkshop install)
Message Text to add to debug console (and file)
Name DebugLog Name used in the SafiWorkshop for the ActionStep
