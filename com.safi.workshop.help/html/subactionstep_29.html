ActionStep

Voicemail

Description

Transfers the call into a given Asterisk voicemail box.

Attributes

Name Default Description
Call1 Call1 The object representing the call to transfer to voicemail
Gain -1 Used to adjust the gain of the recording. The value is an integer representing the amount of gain in decibels.
Mailboxes Voicemail box or boxes to leave a message in
Name Voicemail Name used in the SafiWorkshop for the ActionStep
Play Unavailable Message false ( true | false ) Causes the unavailable message to be played. By default, the message says, "The person at extension 1234 ... is unavailable," but the mailbox owner may record their own unavailable message with the VoicemailMain ActionStep.
Skip Instructions false ( true | false ) Causes the instructions: "Please leave your message after the tone. When done, hang up, or press the pound key." to be skipped.
