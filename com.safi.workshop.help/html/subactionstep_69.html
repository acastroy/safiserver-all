ActionStep

TimeBasedRouting

Description

Takes the appropriate exit path determined by time. To add additional time condition exit paths, select the TimeBasedRouting Action step and click the "Add Time Range" bubble:



To edit a Time Range; double click on the exit path. This will bring up the "Time Range Editor":



The Occurs section selects what interval type to create the time range with (Daily, Weekly, and Monthly). Selecting between these changed the group left of the Occurs group in to the following corresponding interval types:
  • Daily: Interval occurs on a daily or intermittent basis (i.e. everyday or ever other day)
  • Weekly: Weekly interval, select how often by putting in a numberic value for the Every field (i.e. Every 1 Week). Select the days that this interval is active (Monday through Sunday).
  • Monthly: Monthly interval, select either a day of the month (and the frequency) or the day of week radio button (i.e. The 1st Sunday of every month)

The duration is the time in which this event occurs (you can select the from time to time or the All day option). The Starting from value is the date at which to start checking for this condition.
