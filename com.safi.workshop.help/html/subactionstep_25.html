ActionStep

MeetMe

Description

Asterisk Application: MeetMe Conferencing

Conference rooms can be set up using the meetme.conf config file.

For more information: Asterisk config meetme.conf

Attributes

Name Default Description
Admin Mode false ( true | false ) Set admin mode
AGI Script If "Use AGI Script" property is set to true; AGI script to run in the background
Default: conf-background.agi (Note: This does not work with non-Zap channels in the same conference)
Allow Pound User Exit false ( true | false ) Allow attendee to exit the conference by pressing the '#' key
Alone Message Enabled true ( true | false ) If true; the system will play the "you are currently the only person in this conference" message to the first attendee
Always Prompt For Pin false ( true | false ) If true; prompt for the pin even if it is specified
Announce Count false ( true | false ) Announce attendee(s) count on joining a conference
Announce Join Leave false ( true | false ) Announce attendee join/leave with review
Announce Join Leave No Review false ( true | false ) Announce attendee join/leave without review
Call1 Call1 Call Object
Close On Last Marked User Exit false ( true | false ) Close the conference when last marked attendee exits
Conference Number The conference number. If the conference number is omitted, the user will be prompted to enter one.
Dynamically Add Conference false ( true | false ) Dynamically creates conference. Will join if a conference has already been added.
Exit On Extension Entered false ( true | false ) Allows attendee to exit the conference by entering a valid single digit extension of the context specified in ${MEETME_EXIT_CONTEXT} or the current context if that variable is not defined.
Monitor Only Mode fales ( true | false ) Set monitor only mode for current call (Listen only, no talking)
Name MeetMe Name used in the SafiWorkshop for the ActionStep
Pass DTMF false ( true | false ) Pass DTMF through the conference.
Pin Code an attendee needs to enter before being joined to the conference
Play Menu On Star false ( true | false ) Send attendee to menu (user or admin) when the '*' key is pressed
Quite Mode false ( true | false ) Quiet mode (don't play enter/leave sounds)
Record Conference false ( true | false ) Record conference (records as the file name defined in the "Recording Filename" property using format defined in the "Recording Format" property). Default filename is meetme-conf-rec-ConferenceNumber-uniqueid and the default format is wav.
Recording Filename Filename to use if "Recording Conference" is set to true
Recording Format Format to use if "Recording Conference" is set to true
Select Empty Conference false ( true | false ) Select an empty conference
Select Empty Pinless Conference false ( true | false ) Select an empty pinless conference
Talk Only Mode false ( true | false ) Set talk only mode for current call. (Talk only, no listening)
Talker Detection false Set talker detection (sent to manager interface and meetme list)
Use AGI Script false Run the AGI script (defined in the "AGI Script" property) in the background
Use Music On Hold false ( true | false ) Enable music on hold when the conference has a single caller
Video Mode false ( true | false ) Video mode (currently not implemented by Asterisk)
Wait For Marked User false ( true | false ) Wait until the marked user enters the conference
