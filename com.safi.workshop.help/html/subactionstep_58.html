ActionStep

RunQuery

Description

This ActionStep executes a query created using the SQL editor built into the SafiWorkshop. A connection must first be made using the OpenDBConnection ActionStep. After running the RunQuery ActionStep; the results will be populates into the created result set (with the given Result Set Name

NOTE: You must fist enter the Connection property value before selecting a Query.
This ActionStep executes a query created using the SQL editor built into the SafiWorkshop. A connection must first be made using the OpenDBConnection ActionStep. After running the RunQuery ActionStep; the results will be populates into the created result set (with the given Result Set Name

Attributes

Name Default Description
Connection The database connection you want to execute the query using. This must be an opened connection
Name RunQuery Name used by the SafiWorkshop to display the ActionStep
Query The query, set up using the built in SQL editor, to run. Clicking the "..." button will bring up the DB navigator.
Read Only true ( true | false ) Protects the database form updates. Set to false if you wish to preform an update.
Result Set Automatically generated by filling in the Result Set Name property.
Result Set Name This is the name of the result set that this ActionStep will return. After entering a value for this property; a result set object will be automatically generated and put in the Result Set property value.
Scrollable false ( true | false ) Allows navigation within the result set
Variable Name This is the name of the variable to store the number of affect rows in (update query only)
