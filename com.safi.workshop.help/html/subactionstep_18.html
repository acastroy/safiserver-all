ActionStep

StreamAudio

Description

Plays the specified sound file (you need to omit the filename extension). Asterisk sound files are stored by default in the /var/lib/asterisk/sounds directory (the directory path can be changed in asterisk.conf).

Attributes

Name Default Description
Call1* Call1 The object representing the call to play to
Escape Digits # Key user can press to skip this ActionStep
Filename* Name of file to play (with out the extension)
Name* StreamAudio Name used in the SafiWorkshop for the ActionStep
