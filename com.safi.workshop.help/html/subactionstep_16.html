ActionStep

SayPhonetic

Description

Spells the text out using the NATO Phonetic Alphebet

Example: an input of: abc will result in an audio playback of: "alpha bravo charlie".

NATO Phonetic Alphabet (wikipedia)

Attributes

Name Default Description
Call1* Call1 The object representing the call to play to
Escape Digits # Key (or keys) user can press to skip this ActionStep
Name* SayPhonetic Name used in the SafiWorkshop for the ActionStep
Text* String of character that will be played using the NATO Phonetic Alphebet
