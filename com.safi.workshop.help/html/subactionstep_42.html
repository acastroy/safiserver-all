ActionStep

IfThen

Description

ActionStep that takes a boolean value; exits via the appropriate path: true(green) or false(red) paths.

Example Scripting:

condition   	returns true if
----------------------------------------
x == y;            x and y are equal
x != y;             x and y are not equal
x > y; 	        x is greater than y
x >= y; 	        x is greater or equal to y
x < y; 	        x is less than y
x <= y; 	        x is less than or equal to y

Attributes

Name Default Description
Boolean Expression Either a boolean variable or a script that can be evaluated as true/false.

Example:
1 == 2 would return false
NOTE: the text values of true and false may also be used
