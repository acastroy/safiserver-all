ActionStep

PlayTones

Description

Plays a tone list, either the tone named <i>tonename</i> defined in the indications.conf file, or a directly specified <i>tonelist</i> of frequencies and durations. See indications.conf for a description of the specification of a tonelist.

Attributes

Name Default Description
Call1 Call1 The object representing the call to play tones to
Name PlayTones Name used in the SafiWorkshop for the ActionStep
